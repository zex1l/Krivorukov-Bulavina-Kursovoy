# Трекер расходов

Простое веб-приложение для отслеживания личных расходов, ведения бюджета и контроля лимитов по категориям.

## Быстрый старт

### Windows

1. Откройте `run.bat`
2. Приложение автоматически установит зависимости и запустится
3. Откройте ссылку: `http://127.0.0.1:5000`

### macOS / Linux

1. Откройте терминал в папке проекта
2. Выполните:
   ```bash
   chmod +x run.sh
   ./run.sh
   ```
3. Откройте ссылку: `http://127.0.0.1:5000`

### Ручная установка

```bash
# Установка зависимостей
python -m pip install -r requirements.txt

# Запуск приложения
python app.py
```

## Структура проекта

```
cursovoy/
├── app.py                 # основной код Flask
├── database.sql             # структура базы данных
├── requirements.txt       # зависимости
├── run.bat               # скрипт запуска (Windows)
├── run.sh                # скрипт запуска (Unix)
├── README.md             # этот файл
├── expenses.db           # база данных (создаётся при первом запуске)
└── templates/
    ├── layout.html       # базовый шаблон
    ├── index.html        # главная страница
    ├── daily.html        # ежедневный отчёт
    ├── report.html       # месячный отчёт
    ├── categories.html   # управление категориями
    └── edit_expense.html # редактирование расходов
```
